Overview
#

Built a “Harry Potter” themed sorting quiz as a group web project. Quiz takers answer a series of questions and are sorted into one of the four Hogwarts houses based on their responses. All user data and results are stored persistently in a database.

Tech Stack
#

  • Language: PHP, HTML
  • Database: MySQL
  • Methodology: Rapid development (2-week sprint)

Features
#

  • Interactive quiz that sorts users into categories based on their answers
  • Dynamic result pages based on quiz responses
  • Full database integration — all entered information is stored using MySQL
  • Built collaboratively with a team using rapid development techniques

What I Learned
#

  • How to connect a front-end web page to a back-end database using PHP
  • Collaborative web development under a tight two-week deadline
  • Rapid development techniques and working effectively as a team
